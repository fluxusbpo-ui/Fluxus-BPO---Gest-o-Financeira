# Fluxus-BPO - Gestão Financeira

Sistema de gestão financeira (G&P / Fluxus fork). Esse repositório contém os diretórios `frontend/` e `backend/` usados para desenvolvimento local.

Instruções de desenvolvimento (resumo):

Requisitos:
- Node.js (v16+ recomendado)

Instalação e execução (Windows, PowerShell):

```powershell
cd frontend; npm install
cd ..\backend; npm install
```

Execução em desenvolvimento:
- Frontend (porta 3001): `cd frontend; npm run dev`
- Backend (porta 5000): `cd backend; npm run dev`

Execução por clique (Windows):
- Use `run-frontend.vbs` e `run-backend.vbs` para abrir janelas separadas (útil quando o caminho do projeto possui caracteres especiais).

Detalhes e documentação complementar estão em outros arquivos e no diretório `supabase/`.
